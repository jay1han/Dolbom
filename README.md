# Dolbom

This project generates three artifacts:

- An Android app
- A Watchface (under [./Watchface](./Watchface/README.md))
- A Watchapp (under [./Watchapp](./Watchapp/README.md])

To work with the Android app, load the project in Android Studio at the root.
For the Watchface and the Watchapp, please see the respective READMEs.
